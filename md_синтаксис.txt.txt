Основний синтаксис
Це елементи, викладені в оригінальному проектному документі Джона Грубера. Усі програми Markdown підтримують ці елементи.

Елемент	Синтаксис Markdown
Заголовок
# H1
## H2
### H3
Жирний	**bold text**
Курсив	*italicized text*
Блок-цитата	> blockquote
Упорядкований список	1. First item
2. Second item
3. Third item
Неупорядкований список	- First item
- Second item
- Third item
код	`code`
Горизонтальне правило	---
Посилання	[title](https://www.example.com)
Зображення	![alt text](image.jpg)
Розширений синтаксис
Ці елементи розширюють основний синтаксис, додаючи додаткові функції. Не всі програми Markdown підтримують ці елементи.

Елемент	Синтаксис Markdown
Таблиця	| Syntax | Description |
| ----------- | ----------- |
| Header | Title |
| Paragraph | Text |
Огороджений кодовий блок	```
{
  "firstName": "John",
  "lastName": "Smith",
  "age": 25
}
```
Примітка	Here's a sentence with a footnote. [^1]

[^1]: This is the footnote.
Ідентифікатор заголовка	### My Great Heading {#custom-id}
Список визначень	term
: definition
Закреслення	~~The world is flat.~~
Список завдань	- [x] Write the press release
- [ ] Update the website
- [ ] Contact the media
Emoji
(див. також Копіювання та вставка Emoji )	That is so funny! :joy:
Виділіть	I need to highlight these ==very important words==.
